# PCB
OSJ PCB Data
## Specification
### Current Sensing Board
* Power Input - AC 90~240V 50~60HZ
* Main MCU - Atmega328 With Arduino Bootloader
* Current Sensor - Allegro ACS712 Hall Effect Sensor
* Communication - Nordic NRF24L01 2.4Ghz
